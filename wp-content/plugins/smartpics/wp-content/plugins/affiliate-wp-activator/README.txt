A WordPress plugin that provides centralized activation and management functionality for AffiliateWP and its add-ons.

== Description ==

The AffiliateWP & Addons Activator plugin simplifies the management of AffiliateWP and its various add-ons through a centralized activation system. This plugin helps WordPress site administrators efficiently manage their complete AffiliateWP ecosystem.

**Key Features:**
* Centralized activation management for all AffiliateWP add-ons
* Improved error handling with graceful fallback for problematic add-ons
* Full compatibility with WordPress 6.8.1 and PHP 7.4+
* Enhanced performance and reliability
* Auto-update functionality for seamless maintenance

**System Requirements:**
* WordPress 5.2 or higher
* PHP 7.4 or higher
* AffiliateWP base plugin (required)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/affiliate-wp-activator/` directory, or install directly through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure AffiliateWP base plugin is installed and activated

== Frequently Asked Questions ==

= Does this plugin require AffiliateWP to be installed? =
Yes, this plugin is specifically designed to work with AffiliateWP and its add-ons. The base AffiliateWP plugin must be installed and activated.

= Is this plugin compatible with the latest WordPress version? =
Yes, this plugin is tested and fully compatible with WordPress 6.8.1 and is regularly updated to maintain compatibility with new releases.

= What happens if an add-on encounters an error during activation? =
The plugin now includes improved error handling mechanisms that prevent critical errors from disrupting the entire activation process. Problematic add-ons are gracefully handled while allowing other add-ons to continue functioning normally.
